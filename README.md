# To-Do List App

A modern, responsive, and feature-rich To-Do List application built with pure HTML, CSS, and JavaScript.

## Features

- **Add Tasks**: Quickly add new tasks to your list.
- **Edit Tasks**: Double-click a task or use the edit button to modify text.
- **Delete Tasks**: Remove tasks you no longer need.
- **Mark as Complete**: Check off tasks to strike them through.
- **Filtering**: View "All", "Active", or "Completed" tasks.
- **Clear Completed**: One-click removal of all finished items.
- **Data Persistence**: Tasks are saved to your browser's LocalStorage, so you don't lose them on refresh.
- **Dark Mode Design**: A sleek, modern interface with smooth animations.

## How to Use

1. **Open the App**: Simply open `index.html` in any modern web browser.
2. **Add a Task**: Type in the input field and press Enter or click the "+" button.
3. **Manage Tasks**:
   - Click the checkbox to toggle completion.
   - Click the pencil icon (or double-click text) to edit.
   - Click the trash icon to delete.
4. **Filter**: Use the buttons above the list to filter by status.

## Technologies Used

- HTML5
- CSS3 (Variables, Flexbox, Animations)
- JavaScript (ES6+, LocalStorage)

## Setup

No installation required! Just clone the repository or download the files and open `index.html`.
